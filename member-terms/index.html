The Guide Group ガイドメンバー利用規約（内部規約）

ご利用の時間を節約するため、以下に主要なポイントを簡潔にまとめました：

  • 本グループは管理・調整・仲介であり契約当事者ではない
  • メンバーは法令遵守・安全確保の義務を負う
  • 料金体系を遵守し、内部報酬情報を外部開示禁止
  • 依頼者との私下取引・リピーター直接契約は禁止
  • 案件連絡は必ず公式連絡網を使用する

  • 規約違反時は警告・活動停止・除名・損害賠償の対象
  • 本素材（写真・動画等）の利用は原則として本グループ内のみ
  • 除名後は依頼者との再取引禁止

The Guide Group ガイドメンバー利用規約（内部規約）

本規約は、The Guide Group グループ（以下「本グループ」という）が運営するガイド活動に参加する個人（以下「メンバー」という）に適用される内部規則であり、グループの秩序維持・信頼性向上およびリスク回避を目的とする。

第1章　総則

第1条（目的）

本規約は、メンバーが本グループに所属して活動する際の基本的義務、禁止事項、責任分担を明確にし、グループ全体の利益を保護することを目的とする。

第2条（適用範囲）

本規約は、本グループに登録された全メンバーに適用され、メンバーは本規約を遵守する義務を負う。

第3条（定義）

  1. 「本グループ」とは、ガイド案件の管理・調整・仲介を行う団体をいう。
  2. 「メンバー」とは、本グループに登録し、規約に同意したうえで活動する個人をいう。
  3. 「依頼者」とは、本グループを通じてガイド業務を依頼する個人または法人をいう。
  4. 本グループは、依頼者とメンバー間の取引の管理・調整・仲介を行うのみであり、契約の当事者とはならない。
  5. ガイドサービスの履行、法令遵守、安全管理、依頼者との紛争解決その他一切の責任は、当該業務を承諾したメンバーが負うものとする。
  6. 本グループは、依頼者またはメンバー間の債務不履行、事故、損害、怪我、死亡、紛争、苦情その他一切について、一切の責任を負わない。

第2章　本グループの地位と責任制限

第4条（管理・調整・仲介としての立場）

  1. 本グループは、案件の管理・調整・仲介のみを行い、依頼者とメンバーの契約の当事者とはならない。
  2. 本グループは、メンバーの行為や過失、事故、トラブル、損害について一切の責任を負わない。

第5条（免責事項）

  1. 交通事情、自然災害、治安悪化、不可抗力等によって生じた損害について、本グループは責任を負わない。
  2. 依頼者の未払い、キャンセル、クレーム、訴訟その他一切の紛争は、当該メンバーが自己責任と費用において解決する。

第3章　メンバーの資格・義務

第6条（資格条件）

  1. メンバーは、法令を遵守し、必要に応じてガイド関連資格や保険に加入しなければならない。
  2. 品位・信頼を損なう行為を行った場合、本グループはメンバーの資格を停止または除名できる。
  3. メンバーは、依頼者からの問い合わせに対して、速やかに回答する義務を負うものとする。
  4. メンバーが一時的にサービスを提供できない場合、事前に本グループにメールまたはカスタマーサポートを通じて通知するか、行事曆やLiveChatに自動返信を設定するものとする。

第7条（遵法義務）

メンバーは日本国の法令および業務に関する各国の法規を遵守しなければならない。

第8条（安全義務）

依頼者の安全を確保するため、危険行為・過失・不適切な案内をしてはならない。

第9条（料金体系の遵守）

  1. メンバーは本グループが定めた料金体系を遵守し、無断で値引き・上乗せしてはならない。
  2. メンバーは、本グループ内部の報酬体系・配分率・内部価格を依頼者に一切開示してはならない。
  3. 本グループは、依頼者に提供するサービス内容および料金を最終的に決定する権限を有し、必要に応じて現地業者と協議の上でパッケージの設計および価格を調整する。
  4. メンバーは、前項の決定に従い、当該パッケージ内容・価格・販売条件に従うものとし、個別にこれらを変更・約束・提示してはならない。

第4章　業務遂行と禁止行為

第10条（公式連絡）

  1. 案件に関する連絡は必ず本グループ指定の公式連絡網を使用すること。
  2. 非公式連絡に基づくトラブルについて本グループは責任を負わない。

第11条（私下取引の禁止）

  1. メンバーは依頼者と直接取引してはならない。
  2. 本グループを経由せずに顧客から報酬を受領した場合、違約金および損害賠償の対象となる。

第12条（再紹介・リピーター取引の禁止）

  1. メンバーは依頼者から紹介された顧客、または過去に本グループ経由で対応した依頼者と、私下で再度契約してはならない。
  2. リピーター取引も必ず本グループを経由しなければならない。

第13条（外部業者利用）

  1. 包車や第三者業者を利用する場合、本グループに事前申告し、承認を得なければならない。
  2. 無断利用によるトラブルは全てメンバーの責任とする。

第14条（秘密保持）

  1. メンバーは依頼者情報、料金体系、内部資料、グループノウハウを外部に漏洩してはならない。
  2. 違反した場合、本グループは損害賠償請求できる。

第15条（禁止行為）

メンバーは次の行為を行ってはならない。

  1. 本グループを経由しない直接契約
  2. 内部料金・報酬の開示
  3. 依頼者または他メンバーの引き抜き
  4. 虚偽報告・不正精算
  5. 本グループの信用を著しく害する行為

第5章　報告・決済・責任

第16条（報告義務）

案件終了後、メンバーは速やかに報告を行い、精算手続きを完了しなければならない。

第17条（デポジット・残金）

  1. デポジット入金確認前に業務を開始してはならない。
  2. 残金未払いの場合、業務提供を停止できる。
  3. 本グループはエスクローサービスまたは決済代行サービスを提供せず、料金の受領は各メンバーの責任となる。メンバーは、料金の受領および領収書・請求書の発行を自己の責任において行うものとし、本グループはそのための雛形を提供する場合がある。
  4. 受領した料金に関する偽札・不正決済その他のリスクはすべてメンバーが負担し、本グループは顧客からの支払の有効性について一切の責任を負わない。
  5. メンバーは、料金を受領した際には速やかに領収書を提供する義務を負う。
  6. 本グループは、料金の受領または返金に関して発生するいかなる紛争についても一切の責任を負わない。

第18条（デポジット返金時の取り扱い）

  1. お客様がキャンセルした場合、デポジット（予約取扱手数料）については本グループ規定に従い返金されるものとする。
  2. メンバーは、キャンセルによってデポジットが返金された場合、返金分についてメンバーに支払う義務はない。
    これは、メンバーがサービスを提供していないためであり、メンバーの損失は発生しないものとする。
  3. 残金の返金や未払いに関する責任は引き続きメンバーが負うものとし、本グループは一切の責任を負わない。
  4. 本規定はメンバーが本グループに参加する前に明示され、同意の上で活動を行うものとする。

第19条（責任の所在）

  1. メンバーの業務に起因する損害・紛争は、当該メンバーが責任を負う。
  2. 本グループは一切の責任を負わない。

第20条（顧客トラブル・苦情・損害の免責）

  1. 依頼者からの苦情、損害、紛争、事故、怪我、病気、遅延、天災、不可抗力、交通事情等に起因する全ての問題について、本グループは一切の責任を負わない。
  2. 依頼者とメンバー間で発生した紛争や損害に関して、メンバーが責任を負うことを明確にする。
  3. 本グループは、返金、補償、代替手配、損害賠償請求等に関して、顧客への対応義務を負わず、メンバーへの指示・管理も行わない。
  4. この規定は、メンバー及び依頼者に事前に明示され、同意された上でサービスが提供されるものとする。

第6章　懲戒・賠償

第21条（懲戒措置）

  1. 規約違反があった場合、本グループは次の処分を行える。
    1. 警告
    2. 一定期間の活動停止
    3. 除名
  2. 除名後はグループ活動への再参加を一切認めない。

第22条（損害賠償）

  1. 規約違反により本グループに損害を与えた場合、メンバーはその損害を賠償する責任を負う。
  2. 私下取引が発覚した場合、本グループは違約金として相当額（案件総額の2倍以上）を請求できる。

第7章　雑則

第23条（競業禁止）

  1. 本グループが提供または作成した、メンバーのイメージ向上のための写真、動画、宣伝素材（以下「本素材」という）は、
    本グループの活動およびプロモーションのみに使用されるものとする。メンバーは、原則として本素材を本グループ以外のウェブサイト、SNS、広告、その他いかなる媒体でも使用してはならない。
    ただし、メンバー自身の紹介ページや予約ページへの誘導等、本グループのサービス利用促進を目的とした場合はこの限りでない。
  2. メンバーは、本グループと直接的に競合する団体やサービスを無断で運営・支援してはならない。
  3. メンバーは、他のプラットフォームや団体で業務を提供する自由を有し、本グループとの関係に影響を与えることはありません。
  4. 本グループの業務に影響を及ぼす競争行為や利益相反の行為は避ける必要があります。

第24条（除名後の取扱い）

除名されたメンバーは、依頼者の紹介・再取引を一切行ってはならない。

第25条（規約改訂）

本グループは必要に応じて規約を改訂でき、改訂後は全メンバーに直ちに適用される。

第26条（準拠法）

本規約の解釈は日本法に従う。

第27条（裁判管轄）

本規約に関して紛争が生じた場合、本グループの拠点所在地を管轄する日本の裁判所を専属的合意管轄裁判所とする。

附則：本規約は2025年9月1日より施行する。
